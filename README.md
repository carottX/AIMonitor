# AI训练实时监控网站

## 项目结构

- backend/  —— FastAPI后端服务
- frontend/ —— React前端应用

## 快速开始

### 后端
1. 进入 backend 目录，安装依赖：
   ```bash
   pip install -r requirements.txt
   ```
2. 启动服务：
   ```bash
   uvicorn main:app --reload
   ```

### 前端
1. 进入 frontend 目录，安装依赖：
   ```bash
   npm install
   ```
2. 启动开发服务器：
   ```bash
   npm start
   ```
